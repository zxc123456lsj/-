# DeepSeek-V3 技术文档总结

## 1. V3 相比 V2 的改进

- **模型规模与参数**:
  - DeepSeek-V3 是一个强大的 Mixture-of-Experts (MoE) 语言模型，总参数量达到 6710 亿，每个 token 激活 370 亿个参数。

- **架构改进**:
  - 采用了在 DeepSeek-V2 中经过充分验证的多头潜在注意力（MLA）和 DeepSeekMoE 架构。
  - 开创性地提出了一种无需辅助损失的负载均衡策略。
  - 设定了多 tokens 预测训练目标以提升性能。

- **训练效率**:
  - 在 14.8 万亿个多样化、高质量 tokens 上进行了预训练。
  - 训练过程非常稳定，未遇到任何不可恢复的损失激增现象，也未进行任何回滚操作。
  - 完整训练仅需 278.8 百万 H800 GPU 小时。

- **性能表现**:
  - 全面评估表明，DeepSeek-V3 的表现优于其他开源模型，并且性能与领先的闭源模型相当。
  - 在多个基准测试中，如 MMLU-Pro、GPQA-Diamond、MATH 500、AIME 2024、Codeforces 和 SWE-bench Verified 等，DeepSeek-V3 均表现出色。

- **推理与部署**:
  - 提供了高效的推理和部署方案，包括预填充和解码优化。
  - 提出了关于硬件设计的建议，特别是通信硬件和计算硬件方面的优化。

- **数据与超参数**:
  - 数据构建和超参数设置经过精心设计，确保模型在长上下文扩展方面也有出色的表现。

- **FP8 训练**:
  - 采用 FP8 训练，通过混合精度框架、量化和乘法提高精度，同时实现低精度存储和通信，进一步提升了训练效率。
