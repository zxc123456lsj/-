# Stock Volatility Visualizer

> 股票波动率可视化技能 · SKILL.md 完整文档

---

## 一、技能概述

本技能通过 `akshare` 获取股票历史数据，实时计算**日波动率（5日滚动标准差）**和**周波动率（20日滚动标准差）**，将两者叠加绘制在同一张图表中，并根据波动率形态变化生成量化买卖信号建议，最终输出可视化图表 + 中文分析报告。

| 市场 | 代码格式 | 示例 | 说明 |
|------|----------|------|------|
| A股沪市 | 纯6位数字 | `600519` | akshare 直传 |
| A股深市 | 纯6位数字 | `000001` | akshare 直传 |

## 五、核心脚本使用说明

### 5.1 安装依赖

```bash
pip install akshare matplotlib pandas numpy
```

### 5.2 命令行调用

```bash
python stock_volatility_chart.py --code <股票代码> --days <天数> --output <图片路径> --json <信号json路径>
```

#### 参数说明

| 参数 | 必填 | 默认值 | 说明 |
|------|------|--------|------|
| `--code` / `-c` | ✅ | — | 股票代码 |
| `--days` / `-d` | ❌ | `90` | 拉取历史数据天数，建议 60~180 |
| `--output` / `-o` | ❌ | 不保存 | 图片输出路径，如 `result.png` |
| `--json` / `-j` | ❌ | 不保存 | 信号JSON输出路径，如 `signals.json` |

#### 示例

```bash
# 平安银行（000001）近90天分析
python stock_volatility_chart.py --code 000001 --days 90 --output平安银行波动率.png --json signals.json

# 贵州茅台（600519）近180天分析
python stock_volatility_chart.py -c 600519 -d 180 -o 茅台波动率.png
```

### 5.3 作为模块导入使用

```python
from stock_volatility_chart import fetch_stock_data, calculate_volatility, generate_signals, plot_volatility_chart

# 1. 获取数据
df = fetch_stock_data("000001")

# 2. 计算波动率
df = calculate_volatility(df)

# 3. 生成信号
signals = generate_signals(df)

# 4. 绘图
signals = plot_volatility_chart(df, "000001", output_path="output.png")
```

```

│  【上图】股价走势 + 买卖信号标注                          │
```

## 七、买卖信号逻辑

### 7.1 信号体系

| 信号 | 触发条件 | 信号强度 | 含义 |
|------|----------|---------|------|
| 🟢 **买入** | 日波动率接近30日最低 且 周波动率同步低位 | ⭐⭐⭐ | 底部震荡，逢低吸纳窗口 |
| 🟡 **持仓观察** | 价格上涨 > 10日均价10% 但波动率未跟随放大 | ⭐⭐ | 注意短期回调风险 |
| 🔴 **卖出** | 日波动率较前日翻倍 且 vol_ratio > 1.8 | ⭐⭐⭐⭐ | 波动急剧放大，注意止盈 |
| 🚨 **强烈卖出** | vol_ratio > 2.2（日周极度背离） | ⭐⭐⭐⭐⭐ | 风险极高，果断离场 |

### 7.2 核心指标

| 指标 | 公式 | 含义 | 阈值参考 |
|------|------|------|----------|
| **日波动率** | 5日滚动 `std(每日收益率)` | 短期市场情绪 | > 2% 为高波动 |
| **周波动率** | 20日滚动 `std(每日收益率)` | 中期风险中枢 | > 3% 为高风险区 |
| **vol_ratio** | 周波动率 / 日波动率 | 日周背离程度 | > 2 为极端逃顶信号 |

### 7.3 信号 JSON 输出示例

```json

    "signal": "🟢 买入",

    "daily_volatility": 0.9754,
    "weekly_volatility": 1.2132,

  },
  {

    "daily_volatility": 2.451,

  }
```

## 九、数据来源与接口

| 接口 | akshare 函数 | 支持市场 | 更新频率 |
|------|-------------|---------|---------|
| 日线历史 | `ak.stock_zh_a_hist(symbol, period="daily", adjust="qfq")` | A股 | T+1 |
| 港股日线 | `ak.stock_hk_daily(symbol)` | 港股 | T+1 |
| 美股日线 | `ak.stock_us_daily(symbol)` | 美股 | T+1 |

> ⚠️ 所有数据均为 T+1 日收盘数据，不包含盘中实时数据。

---

## 十、注意事项

1. **数据延迟**：akshare 数据为前一交易日收盘价，不含当日盘中实时数据
2. **信号非建议**：波动率量化信号仅供参考，不构成任何投资建议
3. **网络容错**：港股/美股接口可能因网络问题失败，需添加 try-except 兜底
4. **中文字体**：Windows 环境优先使用 Microsoft YaHei，若无则 fallback 到 SimHei
5. **图表保存**：推荐设置 `--output` 参数，图表直接保存为文件便于分享
